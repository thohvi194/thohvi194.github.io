3081W FlashPhoto, Mia and libphoto

Welcome to the homepage for support for both users and developers for our CSCI 3081W project: FlashPhoto, Mia, and libphoto.

FlashPhoto is an image editing application which supports simple drawing and filters, as well as loading and saving images. Mia is a "medical imaging application" used for vieweing medical images such as MRIs or brain scans. 'libphoto' is the base of code we wrote to support all this.

Users

This page provides support for users, located here. It includes all information necessary to get up and running with both FlashPhoto and Mia.

Developers

Interested in using our code, either to expand upon our applications or to write your own? You're in luck. Support for developers can be found here. These pages contain several helpful resources. First is doxygen documentation to understand the APIs in the library we created. Second is a collection of "Design and Coding Style" resources, which include an overview of the project design, tutorials for adding new features, a style/documentation guide, as well as blog posts written by the developers to give some insight into the standard process behind developing our applications.
